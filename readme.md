# 言念起始页 (Yannian Startpage)

> **追求极致纯净与效率的极客浏览器主页。** > 一款专为极客与开发者设计的浏览器起始页，采用 **GUI (可视化)** 与 **CLI (命令行)** 双轨架构，支持无限层级书签管理、备忘录、倒数日以及深度定制的搜索引擎。

---

##  项目亮点

-  **双模切换**：支持现代感十足的“玻璃拟态” GUI 模式与极客味纯正的 CLI 命令行模式，按下 `Ctrl + /` 即可无缝穿梭。
-  **沉浸式终端**：内置强大的 CLI 引擎，支持 Tab 补全、色彩高亮及类 Linux 操作逻辑。
-  **无限层级书签**：抛弃传统的扁平展示，支持虚拟文件树导航，让成百上千个书签井然有序。
-  **聚合搜索**：内置 Bing、GitHub、Bilibili、知乎等 8 大常用搜索引擎，支持自定义扩展。
-  **效率挂件**：集成备忘速记、倒数日阵列以及专属一言库。
-  **隐私至上**：100% 本地存储（LocalStorage），不上传任何用户数据。

##  技术栈

- **HTML5**：语义化结构设计。
- **CSS3**：毛玻璃特效（Backdrop Filter）、响应式布局、Flex/Grid 阵列。
- **JavaScript (ES6+)**：原生 JS 驱动，高性能 DOM 操作，内置终端解析引擎。

##  快速上手

### 1. 浏览器扩展安装 (推荐)
1. 下载本项目代码。
2. 打开 Chrome 或 Edge 的“扩展程序”页面 (`chrome://extensions/`)。
3. 开启 **开发者模式**。
4. 点击 **加载解压的扩展程序**，选择本项目文件夹。
5. 现在，点击“新标签页”即可体验。

### 2. 终端命令手册 (CLI Mode)
进入 CLI 模式后，输入 `help` 即可查看完整命令：
| 命令 | 说明 | 示例 |
| :--- | :--- | :--- |
| `bm ls` | 列出当前目录书签/文件夹 | `bm ls` |
| `[编号]` | 直接输入数字编号打开书签 | `5` |
| `..` | 返回上一级目录 | `..` |
| `search` | 全局搜索 | `search 极客起始页` |
| `time` | 显示当前系统时间 | `time` |
| `note add` | 新增备忘录 | `note add 记得提交代码` |
| `gui` | 返回可视化模式 | `gui` |
